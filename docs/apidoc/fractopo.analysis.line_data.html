fractopo.analysis.line_data module¶

class fractopo.analysis.line_data.LineData(line_gdf: geopandas.geodataframe.GeoDataFrame, azimuth_set_ranges: Tuple[Tuple[float, float], …], azimuth_set_names: Tuple[str, …], length_set_ranges: Optional[Tuple[Tuple[float, float], …]], length_set_names: Optional[Tuple[str, …]])¶

Bases: object

Wrapper around the given line_gdf (trace or branch data).

The line_gdf reference is passed and LineData will modify the input line_gdf instead of copying the input frame. This means line_gdf columns are accesible in the passed input reference.
