fractopo.analysis.general module¶

fractopo.analysis.general.determine_azimuth(line: shapely.geometry.linestring.LineString, halved: bool) → float¶

Calculates azimuth of given line.

If halved -> return is in range [0, 180] Else -> [0, 360]

e.g.: Accepts LineString

>>> determine_azimuth(LineString([(0, 0), (1, 1)]), True)
45.0

>>> determine_azimuth(LineString([(0, 0), (0, 1)]), True)
0.0

>>> determine_azimuth(LineString([(0, 0), (-1, -1)]), False)
225.0

>>> determine_azimuth(LineString([(0, 0), (-1, -1)]), True)
45.0
fractopo.analysis.general.determine_set(value: Union[float, int], value_range: Tuple[float, float], loop_around: bool)¶

Determines if value fits within the given value_range.

If the value range has the possibility of looping around loop_around can be set to true.

>>> determine_set(5, (0, 10), False)
True

>>> determine_set(5, (175, 15), True)
True
