File: contour_grid.py Author: Nikolas Ovaskainen Github: https://github.com/nialov Description: Scripts for creating sample grids for fracture trace, branch and node data

fractopo.contour_grid.create_grid(cell_width: float, traces: geopandas.geodataframe.GeoDataFrame) → geopandas.geodataframe.GeoDataFrame¶

Creates an empty polygon grid for sampling fracture branch data. Grid is created to always contain all given traces.

E.g.

>>> traces = gpd.GeoSeries(
...     [
...             LineString([(1, 1), (2, 2)]),
...             LineString([(2, 2), (3, 3)]),
...             LineString([(3, 0), (2, 2)]),
...             LineString([(2, 2), (-2, 5)]),
...     ]
... )
>>> create_grid(cell_width=0.1, traces=traces).head(5)

fractopo.contour_grid.populate_sample_cell(sample_cell: shapely.geometry.polygon.Polygon, sample_cell_area: float, traces_sindex: geopandas.sindex.PyGEOSSTRTreeIndex, nodes_sindex: geopandas.sindex.PyGEOSSTRTreeIndex, traces: geopandas.geodataframe.GeoDataFrame, nodes: geopandas.geodataframe.GeoDataFrame) → Dict[str, float]¶

Takes a single grid polygon and populates it with parameters usings the other inputs.

E.g.

>>> traces = gpd.GeoDataFrame(
...     {
...             "geometry": [
...                         LineString([(1, 1), (2, 2), (3, 3)]),
...                         ]
...     }
... )
>>> nodes = gpd.GeoDataFrame(
...     {
...             "geometry": [
...                         Point(2, 2),
...                         ],
...             "Class": [
...                         X_node,
...                      ],
...     }
... )
>>> traces_sindex, nodes_sindex = traces.sindex, nodes.sindex
>>> sample_cell = Polygon([(-2, -2), (-2, 2), (2, 2), (2, -2)])
>>> [print(item) for item in populate_sample_cell(

fractopo.contour_grid.sample_grid(grid: geopandas.geodataframe.GeoDataFrame, traces: geopandas.geodataframe.GeoDataFrame, nodes: geopandas.geodataframe.GeoDataFrame) → geopandas.geodataframe.GeoDataFrame¶

Populates a sample polygon grid with geometrical and topological fracture network parameters.

E.g.

>>> traces = gpd.GeoDataFrame(
...     {
...             "geometry": [
...                         LineString([(1, 1), (2, 2), (3, 3)]),
...                         ]
...     }
... )
>>> nodes = gpd.GeoDataFrame(
...     {
...             "geometry": [
...                         Point(2, 2),
...                         ],
...             "Class": [
...                         X_node,
...                      ],
...     }
... )
>>> grid = create_grid(cell_width=0.5, traces=traces)
